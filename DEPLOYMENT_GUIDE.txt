================================================================================
🔧 TROUBLESHOOTING
================================================================================

❌ PROBLEM: Build failed on Render
   ✅ SOLUTION:
   1. Check "Logs" tab in Render dashboard
   2. Common issues:
      - Missing requirements.txt → Ensure file exists
      - Typos in Procfile → Check "web: gunicorn app:app"
      - Git push failed → Re-push code


❌ PROBLEM: App shows 502 Bad Gateway
   ✅ SOLUTION:
   1. Wait 2-3 minutes (free tier cold start)
   2. Check logs for errors
   3. Verify app.py has no syntax errors
   4. Restart service from Render dashboard


❌ PROBLEM: Page loads but answers don't work
   ✅ SOLUTION:
   1. Open browser DevTools (F12)
   2. Check Console for errors
   3. Check Network tab for failed /ask requests
   4. Verify JavaScript is enabled


❌ PROBLEM: Very slow loading (>1 minute)
   ✅ SOLUTION:
   - This is NORMAL for free tier after inactivity
   - First request triggers "cold start" (~50 seconds)
   - Subsequent requests are fast
   - Upgrade to paid tier for always-on service

✅ CORRECT Start Commands:

Production (RECOMMENDED):
    gunicorn app:app

Development (NOT for production):
    python app.py

With workers (for high traffic):
    gunicorn -w 4 app:app

Custom port (if needed):
    gunicorn --bind 0.0.0.0:$PORT app:app


❌ WRONG Start Commands (DON'T use these):

    python app.py                    # No Gunicorn
    flask run                        # Development only
    gunicorn main:app                # Wrong module name
    gunicorn application:app         # Wrong variable name

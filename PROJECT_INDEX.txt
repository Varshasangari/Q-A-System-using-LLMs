  HOW TO USE (Step-by-Step)
================================================================================

1️⃣ OPEN THE APPLICATION:
   - Click the preview button, OR
   - Open browser → http://localhost:5000

2️⃣ ASK A QUESTION:
   - Type your question in the input box, OR
   - Click any example question chip below the input

3️⃣ SUBMIT:
   - Click "Ask Question" button, OR
   - Press Enter key

4️⃣ VIEW ANSWER:
   - Watch the loading animation
   - Answer appears with typing effect
   - Organized in 3 sections:
     * Quick Answer (summary)
     * Detailed Explanation
     * Practical Example

5️⃣ INTERACT:
   - Copy answer (click clipboard icon)
   - Ask another question
   - Or clear everything

================================================================================
  TROUBLESHOOTING
================================================================================

❌ PROBLEM: "Flask not found"
   ✅ FIX: pip install flask

❌ PROBLEM: Page won't load
   ✅ CHECK: Server running? Terminal should show "Running on..."
   ✅ VERIFY: URL is http://localhost:5000
   ✅ TRY: Different browser or hard refresh (Ctrl+Shift+R)

❌ PROBLEM: Port already in use
   ✅ FIX: Change port in app.py line 225
          app.run(debug=True, port=8080)

❌ PROBLEM: Styling broken
   ✅ FIX: Clear browser cache (Ctrl+Shift+Delete)
   ✅ REFRESH: Hard refresh (Ctrl+Shift+R)

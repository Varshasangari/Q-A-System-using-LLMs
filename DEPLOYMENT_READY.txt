🐛 COMMON ISSUES & SOLUTIONS
================================================================================

❌ ISSUE: Build failed on Render
   ✅ CHECK:
   - Logs tab in Render dashboard
   - All files committed to Git
   - requirements.txt exists
   - Procfile exists and says: web: gunicorn app:app

❌ ISSUE: 502 Bad Gateway error
   ✅ WAIT 2-3 minutes (free tier cold start)
   ✅ CHECK logs for actual errors
   ✅ VERIFY Procfile is correct

❌ ISSUE: Page loads but features don't work
   ✅ OPEN browser DevTools (F12)
   ✅ CHECK Console tab for JavaScript errors
   ✅ CHECK Network tab for failed API calls
   ✅ VERIFY static files loaded (style.css, script.js)

❌ ISSUE: Very slow loading (>1 minute)
   ✅ NORMAL for free tier after inactivity
   ✅ First request triggers cold start (~50 seconds)
   ✅ Subsequent requests are fast
   ✅ UPGRADE to paid tier for always-on service
